# MDO3 Digital Portfolio

The main portfolio landing page for MDO3 Digital (mdo3d.com) - showcasing 15+ AI-powered platforms across spiritual guidance, professional tools, utilities, and lead generation.

## About

MDO3 Digital is a division of Lamar (Butts Technologies, LLC), offering a diverse collection of digital platforms:

- **Spiritual & Divination**: Oracle cards, tarot, dreams, numerology, astrology
- **Professional & Career**: Resume analyzer, LinkedIn optimizer, interview prep  
- **Utilities & Productivity**: PDF tools, QR codes, image tools
- **Lead Generation**: Business scanners, email finders, prospect research
- **Plugins & Extensions**: Chrome extensions, Discord bots, Telegram bots
- **Commerce**: Runwae influencer platform

## Tech Stack

- **Framework**: Next.js 15 (Static Export)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel
- **Domain**: mdo3d.com

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm start
```
